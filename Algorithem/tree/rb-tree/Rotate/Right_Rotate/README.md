RB-Tree Right Rotate
-------------------------------------------

#### File list

* rbtree.c

  The core library of rbtree.

* rbtree.h

  The header file of rbtree.

* rb_run.c

  The userspace demo code which descibe how to right rotate rbtree.

#### Usage

Run 'make' command to compile source code, detail as follow:

```
make clean
make
./rbtree
```

Output:

```
0x4 0x5 0x6
```
