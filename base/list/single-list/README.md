Single-list
------------------------------------------

```
Single-list

          +----+       +----+              +----+
          |    |       |    |              |    |
  Head--->|   -|------>|    |----.....---->|    |
          |    |       |    |              |    |
          +----+       +----+              +----+
```


## File Describe

* demo.c

  Simple single-list base on level-double pointer.

* demo1.c

  Advantage demo.c single-list base on level-triple pointer.

## Usage

```
gcc demo.c -o a.out
gcc demo1.c -o a.out
```

## Running

```
./a.out
```
